# Release-finalizer witness ledger

This ledger is an evidence index. A row records only the boundary exercised by
its pointer; it does not turn fake-tool, parser, or host orchestration coverage
into native Windows or cryptographic proof. Box-only result cells remain
placeholders until an operator records the real run.

The canonical source-bound transaction produces the current-only
`target/release-candidate/<VERSION>/` six/seven-artifact bundle plus companion
manifest (seven/eight files total), distinct from the accumulated internal
`Releases/` workspace. Receipts remain outside it under
`target/release-evidence/<VERSION>/`. Every direct publication entry point stays
fail-closed; publication belongs only to the aggregate provenance publisher.
